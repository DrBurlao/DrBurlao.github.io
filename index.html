Our project is a program launcher designed for virtual computer environments or simulators, such as Computercraft in Minecraft. It provides a terminal-based user interface that allows users to access and run programs, utilities, and games efficiently. The launcher utilizes connected peripherals to provide a seamless experience and can detect monitors and modems to extend functionality.

Key Features

  • Peripheral Detection and Connection: The launcher scans for connected peripheral devices and automatically associates with available monitors and modems. This facilitates integration with additional hardware and allows for more dynamic interaction.
  • Menu Navigation: The launcher offers an interactive menu with customizable categories, allowing users to use arrow keys to navigate and the "enter" key to select and run applications. Categories can contain programs, games, file explorers, and other custom applications.
  • Customizable Interface: The launcher's design supports visual elements such as ASCII art, colored bars, and other graphic details to enhance the user experience. It also includes real-time information like an FPS counter and a digital clock to provide useful system status details.

How It Works

The project is built on a main script that performs the following tasks:

  1. Detection of Monitors and Modems: Uses peripheral API functions to search for and connect devices. If a monitor is not found, the script attempts to detect it through modem communication.
  2. Terminal Setup: Sets the active terminal (monitor or default screen) and adjusts text and color settings to optimize the user experience.
  3. Graphical Interface Drawing: The script includes functions to clear the screen and draw elements such as ASCII art, colored bars, text, and more.
  4. Keyboard Event Management: Uses an event loop to listen for keyboard input, allowing menu navigation and option selection.
  5. Run Programs and Scripts: When a user selects a menu option, the script runs the associated program in a new process, enabling multitasking and parallel operations.

Benefits and Uses

  • Centralized Program Access: The launcher simplifies interaction with applications and utilities by grouping them in one place. This improves efficiency and reduces access time.
  • Support for Simulated Environments: Ideal for platforms like Computercraft, where accessing multiple programs can be complicated without a centralized interface.
  • Flexibility and Extendibility: The project is designed to be modular and extendable, allowing the addition of new categories and functionality as needed.
