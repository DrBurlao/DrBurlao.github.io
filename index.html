BurlaOS is a customizable operating system for ComputerCraft, designed to be lightweight and flexible for virtual computer environments in Minecraft. It features a user-friendly terminal interface, peripheral detection, and program launch capabilities.

Key Features

  • Peripheral detection for monitors, modems, and other devices.
  • Interactive menu navigation with keyboard support.
  • Customizable interface with ASCII art, colored bars, and more.
  • Runs programs in a multitasking environment.

To get started, download the latest version of BurlaOS and follow the instructions for installation.
